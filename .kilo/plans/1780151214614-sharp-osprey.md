# Dashboard Typography Revamp

## Analysis

### Current State — Issues

| Issue | Detail |
|-------|--------|
| **Serif font unused** | `DM Serif Display` imported but never used in dashboard UI — only `font-display` token references it |
| **Monospace unused** | `JetBrains Mono` imported but dashboard uses it only for score numbers, inconsistently |
| **All Inter, all the time** | Headings, body, UI all use Inter — no typographic differentiation |
| **No weight scale** | 700 and 800 used interchangeably with no hierarchy rules |
| **Inconsistent sizing** | Headings mix `clamp()`, `px` values, and Tailwind classes with no pattern |
| **Inline styles dominate** | 80% of font declarations are inline `style={{ fontSize, fontWeight }}` — unthemeable |
| **Letter-spacing chaos** | Values range from `-0.04em` to `0.12em` with no rules |
| **Mono mismatched** | JetBrains Mono 400/500/600 weights loaded, but only `font-extrabold` used |

### Desired State — Claude-inspired

```
┌─ Display / Major headings ─── DM Serif Display (Georgia fallback) ─┐
│  Dashboard title, score cards, lens header, report title           │
├─ Body / UI ────────────────── Inter (system-ui fallback) ──────────┤
│  Sidebar labels, form text, metric labels, descriptions            │
├─ Mono / Data ───────────────── JetBrains Mono ─────────────────────┤
│  Score numbers, percentages, addresses in monospace                │
└────────────────────────────────────────────────────────────────────┘
```

### Typography Scale

| Token | Size | Weight | Usage |
|-------|------|--------|-------|
| `text-hero` | 28-36px | 700 (serif) | Dashboard greeting |
| `text-heading` | 20-24px | 700 (serif) | Section titles |
| `text-subheading` | 16-18px | 600 (sans) | Card titles, sidebar headers |
| `text-body` | 14-15px | 400 (sans) | Descriptions, messages |
| `text-caption` | 11-12px | 500 (sans) | Labels, metadata, timestamps |
| `text-mono` | 12-14px | 500 (mono) | Score numbers, data |
